# damage-logic.md

## 목적

이 문서는 카드 이벤트 큐 기반 전투에서 피해, 방어, 대상, 행동 순서를 계산하는 기본 규칙을 정의한다.

## 핵심 구조

각 파티원은 개별 덱과 손패를 가진다. 플레이어는 파티원을 선택해 해당 파티원의 손패를 확인하고, 파티 전체가 공유하는 코스트를 소비해 카드를 사용한다.

카드는 전투 중 즉시 결과를 만드는 대신, 기본적으로 `Action Event`를 생성한다.

```text
카드 사용
-> Action Event 생성
-> 플레이어 행동 큐에 등록
-> 턴 종료
-> 큐 순서대로 행동 실행
-> 피해/방어/상태이상 처리
```

단, 카드 설계상 즉시 처리되어야 하는 드로우, 에너지 회복, 카드 생성 등은 `Instant Effect`로 분류할 수 있다.

## 덱과 코스트 규칙

| 항목 | 규칙 |
| --- | --- |
| 덱 | 파티원별 개별 덱 |
| 손패 | 파티원별 개별 손패 |
| 코스트 | 파티 전체 공용 코스트 |
| 카드 사용 | 선택한 파티원의 손패에서 사용 |
| 행동 주체 | 기본적으로 해당 카드를 보유한 파티원 |
| 행동 큐 | 모든 파티원의 사용 카드가 하나의 플레이어 행동 큐에 합쳐짐 |

예시:

```text
공용 코스트 3
수호자 카드 1장 사용: 코스트 1
척후병 카드 1장 사용: 코스트 2
남은 코스트 0
턴 종료 후 수호자 이벤트 -> 척후병 이벤트 순서로 실행
```

## 카드 분류

| 분류 | 설명 | 사용 기준 |
| --- | --- | --- |
| Common Card | 모든 직업이 사용할 수 있는 공용 카드 | 기본 공격, 기본 방어, 단순 유틸리티 |
| Class Card | 특정 직업 또는 파티원 역할 전용 카드 | 수호자 방어, 암살자 연타, 사제 회복 |
| Curse Card | 덱과 전투 선택지를 방해하는 카드 | 사용 불가, 코스트 낭비, 피해 유발 |
| Special Card | 특정 적 기믹, 이벤트, 보스전에서 사용 가능한 카드 | 보스 장치 해제, 필드 상호작용 |

추후 카드 분류는 추가될 수 있다. 단, 새 분류는 기존 분류로 표현할 수 없는 플레이 목적이 있을 때만 추가한다.

## 이벤트 타입

| 타입 | 설명 | 예시 |
| --- | --- | --- |
| Attack | 피해를 준다 | 단일 공격, 광역 공격, 연타 |
| Guard | 방어도를 얻는다 | 자신 방어, 아군 보호, 파티 방어 |
| Skill | 전투 보조 효과 | 카드 드로우, 에너지 회복, 카드 강화 |
| Status | 상태이상을 부여한다 | 취약, 약화, 표식, 기절 |
| Support | 아군을 강화한다 | 힘 증가, 보호막, 다음 공격 강화 |
| Special | 캐릭터 고유 규칙 | 반격 태세, 소환, 위치 변경 |

## 기본 피해 공식

```text
최종 피해 = max(0, 기본 피해 + 공격자 공격 보정 + 카드 보정 - 대상 방어 보정)
```

초기 프로토타입에서는 계산식을 단순하게 유지한다.

```text
카드 피해 = 카드 기본값 + 파티원 Power
실제 체력 피해 = max(0, 카드 피해 - 대상 Block)
```

## 방어 규칙

```text
획득 방어도 = 카드 기본 방어 + 파티원 Guard Power
```

기본값:

- 방어도는 파티원별로 보유한다.
- 턴 종료 정리 단계에서 일반 방어도는 0으로 초기화된다.
- 특정 카드나 상태는 방어도 유지 규칙을 예외로 만들 수 있다.
- 카드 효과로 모든 파티원의 방어도를 올릴 수 있다.
- 카드 효과로 특정 파티원이 아군이 받을 피해를 대신 받을 수 있다.

## 피해 대신 받기 규칙

피해 대신 받기는 파티원별 방어도를 유지하면서도 탱커 역할을 만들기 위한 예외 규칙이다.

```text
피해 대상 확인
-> 피해 대신 받기 효과 확인
-> 대신 받을 파티원으로 피해 대상 변경
-> 대신 받는 파티원의 Block과 상태를 기준으로 피해 계산
```

동시에 여러 대신 받기 효과가 있을 경우 우선순위는 아래 순서를 따른다.

1. 카드에 명시된 전용 보호 효과
2. 가장 최근에 적용된 보호 효과
3. 체력이 가장 높은 보호자

## 대상 규칙

카드는 아래 대상 타입 중 하나를 가진다.

| 대상 타입 | 설명 |
| --- | --- |
| Self | 행동하는 파티원 자신 |
| All Allies | 모든 아군 |
| Target Ally | 선택한 아군 1명 |
| Target Enemy | 선택한 적 1마리 |
| All Enemies | 모든 적 |
| All Units | 모든 아군과 모든 적 |
| Field | 필드 또는 전장 상태 |
| Random Enemy | 무작위 적 1마리 |
| Lowest HP Enemy | 체력이 가장 낮은 적 |
| Marked Target | 표식이 있는 대상 우선 |

## 행동 주체 규칙

카드 이벤트는 반드시 `행동 주체`를 가진다.

기본 규칙:

- 카드는 파티원 개별 덱에 들어간다.
- 카드를 사용한 파티원이 행동 주체가 된다.
- 공용 카드는 여러 직업 덱에 들어갈 수 있지만, 실제 행동 주체는 그 카드를 사용한 파티원이다.
- 직업 전용 카드는 해당 직업 또는 파티원만 덱에 넣을 수 있다.
- 저주 카드는 행동 주체가 없거나, 보유 파티원에게 부정적 효과를 준다.
- 특수 카드는 적 기믹 또는 필드 조건이 맞을 때만 사용 가능할 수 있다.

## 실행 순서

플레이어 행동 큐는 카드 사용 순서대로 실행된다.

```text
1번 카드 이벤트
-> 2번 카드 이벤트
-> 3번 카드 이벤트
-> 플레이어 행동 종료
-> 적 AI 행동 시작
```

예외 규칙:

- `Fast`: 큐 앞쪽으로 이동한다.
- `Delayed`: 다음 턴 시작 시 실행된다.
- `Reaction`: 조건 충족 시 적 턴 중 끼어든다.
- `Combo`: 직전 이벤트 타입을 참조한다.

예외 규칙은 재미를 만들 수 있지만 복잡도를 크게 올리므로, MVP에서는 `Combo` 1종만 제한적으로 사용한다.

## 예시 카드 처리

```text
카드: 방패 올리기
주체: 수호자
타입: Guard
대상: Self
효과: Block 8 획득
```

```text
카드: 찌르기
주체: 척후병
타입: Attack
대상: Enemy
효과: 피해 6
```

사용 순서:

```text
방패 올리기 -> 찌르기 -> 턴 종료
```

실행 결과:

```text
수호자 Block 8 획득
척후병이 선택한 적에게 피해 6
적 AI 행동
```

## 미정 항목

- Power, Guard Power 같은 스탯을 파티원별로 둘지 카드 태그별로 둘지
- 공격 이벤트가 대상 사망 시 자동으로 다른 적을 재지정하는지
- 카드 이벤트 큐를 UI에서 얼마나 상세히 보여줄지
- 공용 카드를 모든 직업 덱에 자유롭게 넣을 수 있는지, 직업별 허용 풀이 있는지
- 특수 카드가 보상으로 획득되는지, 전투 중 임시 생성되는지
